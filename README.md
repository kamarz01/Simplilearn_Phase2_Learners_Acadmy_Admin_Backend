# Simplilearn_Phase2_Learners_Acadmy_Admin_Backend

## Intro

Practise Project: Learning Academy Admin Backend System.

The project was part of Simplilearn **Full Stack Java Developer Master Program** Phase (2).

## Technologies
- Java EE
- Servlets
- Hibernate
- MySql
- Java Streams
- JSP
- Tomcat Server 8.5
- HTML5
- CSS3
- AdminLTE UI
